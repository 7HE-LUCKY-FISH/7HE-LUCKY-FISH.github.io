A web application that calculates the optimal economy strategy for the game Bloons Tower Defense 6. The application allows users to input their current game state and receive the optimal strategy to maximize their in-game currency. This is optimized for CHIMPS and IMPOPPABLE game modes.

MTG MarketPlace Bot/Web App

Python Postgres AWS

Developed a chatbot using Python, allowing users to look up cards and game interactions. Utilizing Scryfall API to access their card database. The bot is currently being utilized by 5 servers and has over 200 users. Other commands such as dice rolling and coin flipping are also available to streamline the start of games. Has a web app connected to it to allow users to track their win/loss and previous matches.
