Major Map

React AWS PyTorch

Major Map is a comprehensive web application designed for San Jose State University (SJSU) students to streamline the complex process of long-term academic planning and semester scheduling. By consolidating fragmented information into a single platform, the application helps students stay on track for timely graduation.

CampusNest

React ExpressJS MongoDB

Won at SpartUp Hackathon 2025. A web application that allows students to find and connect with other students in their dorms. The application allows users to create profiles, post messages, and search for other students based on their interests. The application is designed to help students make friends and find study partners in their dorms.

Groecery/Shopping Website

Python/Flask React MySQL

This project was a demo in designing and developing a comprehensive e-commerce website for "OFS," a local organic food retailer in San Jose Downtown. This was the demo project for the club to evaluate if having a website would be beneficial for the club business.

SOMS Sports Organization Managment System

Python/Flask React MySQL

A sports management system for soccer teams to help manage their players and schedules. It allows coaches to create formation and see injury reports in real times from medical staff. It also allows for its network of scouts to find players and put them in the system.
